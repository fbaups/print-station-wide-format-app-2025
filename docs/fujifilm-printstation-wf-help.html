FUJIFILM PrintStation WF -help

What to do when things go wrong?

Jobs not arriving? Firstly, if jobs are not arriving into the FUJIFILM PrintStation WF interface restart the FUJIFILM desktop computer. Check that other devices on the network are working and that the IK Server is running.

Jobs not releasing to printer? If jobs are arriving at the IK PrintStation WF interface but you are unable to release them, check that the printer is powered on and has ink and paper. Then reboot the IK PrintStation computer.

Can't Login? Restart the computer and try again, be sure that the CAPS lock button is not on.

Jobs won't delete? The computer will auto delete jobs beyond a certain age.

Can't find a job in the history? Jobs will delete automatically beyond a certain age.

Job is incorrectly scaled or rotated? Use the edit function to adjust the layout and re-print.
