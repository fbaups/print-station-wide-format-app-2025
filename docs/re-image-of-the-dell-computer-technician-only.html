FUJIFILM PrintStation WF Windows computer desktop restore

Re-Image of the Computer (Technician only)

In the case of catastrophic failure of Hard Disk or Operating System, the fastest way to return to printing is to re-image the Dell system.

Follow the below process

Procedure

Insert Windows system restore USB and reboot from USB. Follow the instructions on the USB stick.

Once the desktop computer has been restored to operating default condition please check that the IP addresses of both network cards are configured correctly and that jobs are arriving from the IK server and printing to the Epson Wide Format printer.
